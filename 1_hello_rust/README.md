## What is Rust

## A Rust program that prints "Hello Rust!"

```rust
//main.rc
fn main() {
  println!("Hello Rust!");
}
```

'!' means that it is a Macro (and not a function).
See the official [document](https://doc.rust-lang.org/book/ch19-06-macros.html) about Macros.

## Usage
```bash
sudo apt install rustc
rustc main.rc
./main
```

## Result
```bash
Hello Rust!
```
